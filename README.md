# Hospital Management System Backend

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Features
- Authentication System
- Patients Management
- Doctors Management
- Appointments Management
- Billing Management
- Rooms Management
- Admissions Management

## Installation

```bash
npm install
```

## Setup Environment

Create `.env` file:

```env
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/hospital_management
JWT_SECRET=supersecretkey
```

## Run Backend

```bash
npm run dev
```

## API Base URL

```bash
http://localhost:5000/api
```
